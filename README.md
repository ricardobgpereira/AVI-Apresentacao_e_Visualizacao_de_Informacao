# Student Performance Dashboard

**Course:** Information Visualization and Presentation
**Final Grade:** 16/20

## Project Overview

This project focuses on the design and development of interactive dashboards to analyze student performance and behavior patterns. The goal is to transform raw educational data into actionable insights for decision-making.

The dataset used is available on Kaggle:
[https://www.kaggle.com/datasets/dipam7/student-grade-prediction/data](https://www.kaggle.com/datasets/dipam7/student-grade-prediction/data)

---

## Objectives

* Analyze student performance and behavioral factors
* Identify patterns affecting academic results
* Build interactive dashboards to support decision-making
* Tailor insights for different stakeholders (personas)

---

## Personas & Use Cases

### Minister of Education

The dashboard supports high-level decision-making by:

* Identifying trends in student performance across demographics
* Analysis of alcohol consumption patterns (weekday vs weekend)
* Relationship between social activity and academic performance
* Understanding the impact of social behavior (e.g., alcohol consumption, social activity)
* Supporting policy creation to improve educational outcomes

### Teacher

The dashboard enables teachers to:

* Monitor student performance and identify at-risk students
* Analyze how lifestyle factors influence academic results
* Grade distribution across different periods
* Performance comparison by age
* Interest in higher education
* Identification of performance trends
* Support data-driven decisions to improve teaching strategies and student outcomes

## Tools & Technologies

* **Power BI** – Dashboard development and visualization
* **PowerQuery** – Data cleaning and transformation
* **Visualization Techniques** – Interactive charts, filters, and storytelling

---

## Key Insights

* Social behavior (e.g., frequency of going out, alcohol consumption) shows a noticeable impact on student performance
* Certain age groups demonstrate higher academic averages
* Clear patterns emerge that can help identify students at risk

---

## Key Learnings

* Designing dashboards for different stakeholders
* Translating data into actionable insights
* Applying visualization best practices
* Supporting decision-making through data storytelling
